使用須知

本系統僅作為個資協同檢查之輔助工具，掃描結果無法保證 100% 偵測所有疑似個資，仍可能存在漏判或誤判，不可僅憑系統結果即認定資料合規或無虞。

  • 資料正式上傳、發布或對外提供前，務必由至少兩人覆核（雙人覆驗）確認內容。
  • 網頁頁尾資料維護、網頁建置、承辦人等依法須公開之聯絡姓名，系統預設不列入個資告警。
  • 資訊處僅提供本系統之檢測服務；最終是否涉及個資、是否可對外使用，以及相關法遵責任，仍由資料提供單位及經辦人員自行負責。

個資 (PII) 自動掃描器

支援台灣身分證、手機、信用卡、Email、統編、車牌、IP、護照、健保卡號等格式

管理白名單 →（需設定 PII_ADMIN_PASSWORD）

勾選後會額外呼叫 Azure AI Language PII API，加強姓名等語意偵測。預設關閉以控制預算；單次最多分析 5000 字。

支援格式：文字檔（.txt .csv .json .html .log .md .xml 等）、Excel（.xlsx .xlsm .xls，逐工作表掃描）、PDF（.pdf，逐頁掃描）、開放文件（.ods .odt）、Word（.docx）。掃描影像 PDF（無文字層）目前不支援。

可輸入一般網頁，或直接貼上 PDF / Word (.docx) / Excel 下載連結。結果會依頁面或下載檔彙總，文件格式會標示工作表或頁碼。

爬取 HTML 頁面時會跟進 PDF / Word / Excel 下載連結。報告依各 URL / 下載檔列出命中摘要；無法抓取或解析的連結會單獨列出。
